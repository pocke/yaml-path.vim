YAML-path.vim
=====

Display the path of YAML on cursor.

![peek 2017-12-09 12-54](https://user-images.githubusercontent.com/4361134/33792358-6e3bbc52-dce0-11e7-8c74-a5701a431391.gif)

Usage
-----

```vim
:call yaml_path#show()
```

Requirements
---

- `+if_ruby`
  - Ruby 2.3 or higher
  - Psych 3.0.0 or higher
